# AI-Powered Clinical Decision Support System

A Safety-Sensitive and Uncertainty-Aware Clinical Decision Support Framework using BioBERT and Retrieval-Augmented FLAN-T5.

## Features
- BioBERT disease prediction
- XGBoost ensemble learning
- Monte Carlo Dropout uncertainty estimation
- Temperature scaling calibration
- Retrieval-Augmented Generation (RAG)
- FLAN-T5 treatment generation
- RxNorm Drug Interaction validation
- Antibiotic stewardship safety checking
- Gradio-based web interface

## Technologies Used
- Python
- PyTorch
- Transformers
- XGBoost
- Gradio
- BioBERT
- FLAN-T5
- Gemini API
